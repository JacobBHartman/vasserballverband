The purpose of this page is to demonstrate an example application that builds off of the wasserballverband API. Links above include the codebase, a link to the API, and a link to a publically editable database which fuels the API data.

There are numerous apps that can be built off the API...

  • Top 10 Lists based on City, County, Regions, States, NCAA, High School, Club
  • Best cities for water polo based on participation rates and rankings
  • Pool Cost comparison chart
  • Water Polo Wiki
  • Mobile Apps
